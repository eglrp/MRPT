=head1 NAME

map-partition - Spectral bisection of observation graphs

=head1 SYNOPSIS

map-partition I<input_map.simplemap> [I<nCut threshold>]

=head1 DESCRIPTION

B<map-partition> takes an input map and generates a set of sub-maps using 
recursive spectral bisection with a threshold of 1 (default), or the one passed
as second argument.
