=head1 NAME

rawlog2carmen - Convert robotic datasets between CARMEN and Rawlog formats

=head1 SYNOPSIS

rawlog2carmen -i I<in_file.rawlog> -o I<out_file.log>

rawlog2carmen --help

=head1 DESCRIPTION

B<carmen2rawlog> is a command-line application to convert robotic datasets 
from the "rawlog" standardized file format to the CARMEN format.
