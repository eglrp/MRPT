=head1 NAME

hmt-slam-gui - Robotic Hybrid Metric Topological (HMT) mappping GUI

=head1 SYNOPSIS

hmt-slam-gui

=head1 DESCRIPTION

B<hmt-slam-gui> is a GUI application which allows loading, editing and building
complex HMT maps for robotics application.
